# 数据结构提点

## 顺序结构

### 表结构

#### 基本概述：

除了第一个元素和最后一个元素，每个数据元素只有一个前驱数据元素和一个后继数据元素

第一个元素没有前驱元素

最后一个元素没有后继元素

#### 顺序表：

##### 基本定义：

顺序表（Sequential List）是线性表的一种存储结构，它使用**一段地址连续的存储单元**依次存储线性表的数据元素，当空间不足时，会自动扩容。

##### 数据类型的定义:

```java

//数据空间的引用 (指针)

//最大容量
int capacity;
//当前长度 最后指向位置
int len;
```

#### 基本功能的实现：

##### 初始化顺序表：

##### 扩容方法

```java
public void enlarge(ArrayList arrayList)
{
    //扩容到当前capacity的两倍
    int[] newData=new int[capacity*2];
    this.capacity*=2;
    for(int i=0;i<len;i++)
    {
        //把原来数组中的元素赋值到新数组中
        newData[i]=data[i];
        //这里也可以用arraycopy代替
        System.arraycopy(data,0,newData,0,len);

    this.data=newData;
}
```

```java
public void push_back(int val)

    data[pos]=val;
    len++;
}
```

#### 链表

##### 基本概述

链表（Linked List）是一种**线性数据结构**，它通过**指针**将一组**零散的内存块**串联起来。每个内存块称为一个**结点（Node）**。

```java
{
```

```java
    Node newNode = new Node();
    newNode.data = val;
    newNode.next = pre.next;
    pre.next = newNode;

    if(newNode.next==header.head)
```

```java
package LinkList;
/*
约瑟夫环 没有头结点的链表 指针直接指向首元节点
 */
public class Joash
{
    class node
    {
        int pos;
        node next;
    }
    //约瑟夫环的head指向首元节点 也就是没有头结点
   class header
   {
       node head;
       node tail;
   }
   header header=new header();
    public int n;
    Joash(int n)
    {
        header.head=null;
        header.tail=null;
        this.n = n;
        for(int i=1;i<=n;i++)
        {
            node newNode = new node();
            newNode.next=header.head;
            newNode.pos = i;
            if(i==1)
                header.head=header.tail=newNode;
            else
            {
                //初始化 模拟尾插
                header.tail.next=newNode;
                header.tail=newNode;
            }
        }
    }
    //拓展 需要展示/遍历这种链表的方案
    public void show()
    {
        node point = header.head;
        do//先执行一次 当然用n判断也行
        {
            System.out.print(point.pos+" ");
            point = point.next;
        }while(point!=header.head);
    }
    public void delete(node pre)
    {
        if(pre.next.next==header.head)
            header.tail=pre;
        pre.next = pre.next.next;
        --this.n;
    }
    public void start(int k)
    {
        node point = header.head;
        while(true)
        {
            if(n==1)
            {
                System.out.println(point.pos);
                break;
            }
           for(int i=1;i<k-1;i++)
           {
               //这个范围的意义 移动的次数等于间隔数-1
               //要找到pre元素 所以再-1 
               //总共-2
               point=point.next;
           }

        }
    }
```

##### (Stack)栈

###### 基本概述: 

栈是一种先进后出的数据结构,在回溯,搜索等算法中有很重要的作用,按照存储方式分为顺序栈和链式栈

###### 顺序栈:

比较简单，直接贴代码了

```java
package Stack;
/*
满递增顺序栈
 */
public class Stack<T>
{
    final int MAXSIZE=10;
    /* 数据结构的定义 */
    public int top;//栈顶指针,指向最后插入的元素
    //这里注意 泛型变量不能实例化啊
    public Object []data=new Object[MAXSIZE];
    /* 初始化栈 */
    Stack()
    {
        //因为是满递增栈，所以栈顶指针先加后存，开始就是-1没问题
        this.top=-1;
    }
    /* 压栈 */
    public void push(T val)
    {
        //爆了
        if(top>=MAXSIZE-1)
            System.out.println("Stack full");
        else
            data[++top]=val;
    }
    /* 弹栈 */
    public T pop()
    {
        //空了
        if(top==-1)
        {
            System.out.println("Stack empty");
            return null;
        }
        else
            return (T)data[top--];
    }
    /* 查看栈顶元素 */
    public T getTopStack()
    {
        return (T)data[top];
    }
}
```

###### 链式栈

```java
package Stack;
/*
链式栈
 */
public class LinkStack<T>
{
    /* 数据结构定义 */
    class node
    {
        T data;
        node next;
    }
    /* 表头 */
    class header
    {
        node top;
        int count;
    }
    header head = new header();
    /* 初始化 */
    LinkStack()
    {
        head.top=null;
        head.count=0;
    }
    /* 压栈 */
    public void push(T val)
    {
        node newnode = new node();
        newnode.data=val;
        newnode.next= head.top;
        head.top=newnode;
        ++head.count;
    }
    //压栈比较新奇
    /* 因为我们采用单向链表实现栈，如果弹栈的时候我们是正序存储的话就无法找到上一个元素了，而我们又	    并不想使用双向链表,所以在存储元素时,就需要能够找到上一个存入的元素,所以我们采用一种前置插入		  的方式
    */   
    /* 弹栈 */
    public T pop()
    {
        if(head.top==null)
        {
            System.out.println("Stack empty!");
            return null;
        }
        else
        {
            node temp=head.top;
            head.top=head.top.next;
            --head.count;
            return temp.data;
        }
    }
    /* 查看栈顶元素 */
    public T getTopLinkStack()
    {
        if(head.top==null)
            return null;
        return head.top.data;
    }
}
```

一般而言有两种方案：

1.搞一个计数器(不常用) 计算数量

2.牺牲一个空间，让空和满的状态区分开

```java
//满：
```

```java
package Queue;
/*
顺序队列 浪费一个空间的做法
先加再出
front永远指向待出列元素的前一个位置
rear永远指向队尾元素
先出再加
front永远指向待出列元素
rear永远指向队尾元素的下一个位置(待插入位置)
环形队列如果不留空白 头尾重合就会出现空和满这两种情况
而留空白之后  空白始终夹在满队列的头尾之间，只有一开始是重合，这样就能区分了
 */
public class ArraysQueue<T>
{
    /* 数据结构的定义 */
    //最大容量
    final int MAXSIZE = 4;
    //数据数组
    Object [] data=new Object[MAXSIZE];
    //指向第一个元素的索引
    int front;
    //指向最后一个元素的索引
    int rear;
    /* 初始化顺序队列 */
    ArraysQueue()
    {
        front=rear=0;
    }
    /* 判断队列是否为空 */
    public boolean isEmpty()
    {
        return front==rear;
    }
    /* 判断队列是否为满 */
    public boolean isFull()
    {
        return (rear+1)%MAXSIZE==front;
    }
    /* 插入元素 */
    public int addQueue(T val)
    {
        //插入失败
        if(isFull())
            return 0;
        else
        {
            //先赋值再加 指向的是插入位置
            data[rear]=val;
            rear=(rear+1)%MAXSIZE;
            return 1;
        }
    }
    /* 删除元素 */
    public int outQueue()
    {
        if(isEmpty())
            return 0;
        else
        {
            //出队列也是先出再加
            data[front]=null;
            front=(front+1)%MAXSIZE;
            return 1;
        }
    }
    /* 获取队列首元素(但不删除) */
    public T getElement()
    {
        return (T)data[front];
    }
}
```

```java
链式队列
和链式栈不同的是，链式队列是尾插

public class LinkQueue<T>
{
    /* 数据结构定义 */
    class node
    {
        T data;
        node next;
    }
    /* 表头定义 */
    class header
    {
        node front;
        node rear;
        int num;
    }
    header head = new header();
    /* 初始化一个链式队列 */
    LinkQueue()
    {
        head.front = null;
        head.rear = null;
        head.num = 0;
    }
    /* 插入 */
    public int addQueue(T val)
    {
        node newNode = new node();
        newNode.data = val;
        //newnode就是最后一个元素
        newNode.next=null;
        //如果插入的是第一个元素才需要变更front
        if(head.front == null)
        {
            head.front=newNode;
            head.rear=newNode;
        }
        //否则只需要变更rear
        else
        {
            head.rear.next=newNode;
            head.rear=newNode;
        }
        head.num++;
        return 1;
    }
    /* 出列 */
    public T OutQueue()
    {
        if(head.front == null)
        {
            head.rear=null;
            head.num--;
            return null;
        }
        node temp = head.front;
        head.front = head.front.next;
        head.num--;
        return temp.data;
    }
    /* 删除所有 */
    //Tips:JVM的垃圾回收机制,当某个对象无法被任何直接引用或者任何间接引用找到时,JVM就会自动回收
    public void deleteAll()
    {
        head.front = null;
        head.num=0;
    }
}
```

##### (Tree)树

###### 基本定义：

每个元素只有一个前驱元素，但可以有很多个后继元素

###### 基本概念：

1.**节点**：树中存储的每一个元素都可以称作节点

​	1.1**根节点**：一个节点没有前驱节点，该节点就称为根节点

​	1.2**父节点**：一个节点的直接前驱节点称为该节点的父节点

​	1.3**子节点**：一个节点的直接后继节点称为该节点的子节点

​	1.4**兄弟节点**：一个节点的父节点下所有除自己之外的节点称为该节点的兄弟节点

​	1.5**叔叔节点**：一个节点的父节点的所有兄弟节点称为叔叔节点

2.**节点的度**：一个节点后有多少个子节点就成为该节点的度

3.**树的度**：一个树中所有节点度的最大值称为树的度

4.**叶子节点**：一个节点没有后继节点，称为叶子节点

5.**树的深度**：根节点为第一层，向下递增，到叶子节点为止的层数称为树的深度

###### 存储方法：

1.双亲存储法：

双亲表示法采⽤顺序表（也就是数组）存储普通树，其实现的核心思想是：顺序存储各个节点的同 时，给各节点附加⼀个记录其父节点位置的变量。 根节点没有父节点，因此根节点记录父节点位置的变量通常置为 -1

###### BFS(广度优先搜索)

1.概述：广度优先搜索，顾名思义，就是要以广度为首要进行遍历，在二叉树中，需要遍历完整个层次后再进入下一层，也就是像雷达一样

2.算法设计：由于我们所能看到的任务增长速度远大于单次处理任务的速度，所以需要缓存任务，而且需要先发现的先处理，很自然的使用队列进行缓存(先进先出)，具体方法就是，处理A任务---->在A任务中发现B和C任务存入队列，A出列，以此类推，在处理所有节点时，节点的左右两侧节点都会存储到队列中，直到处理的元素个数等于存入的元素个数(整个树处理完毕)，树就被广度遍历完了，队列(缓存区)也就空了

3.代码实现：

```java
/* 对节点的访问 */
public void visitNode(TreeNode node)
{
    System.out.print(node.data+" ");
}
/* 广度优先搜索 */
//核心思想 缓存思想 拿队列进行缓存
public void BFS()
{
    int MaxSize = 10;
    TreeNode[] nodes = new TreeNode[MaxSize];
    int rear=0;int front=0;
    //初始化任务,不然一开始就空了
    nodes[rear]=root;
    rear=(rear+1)%MaxSize;
    while (rear != front)//空队列
    {
        visitNode(nodes[front]);
        if (nodes[front].left != null) {
            nodes[rear] = nodes[front].left;
            rear = (rear + 1) % MaxSize;
        }
        if (nodes[front].right != null) {
            nodes[rear] = nodes[front].right;
            rear = (rear + 1) % MaxSize;
        }
        front = (front + 1) % MaxSize;
    }
}
```

###### DFS(深度优先搜索)

1.概述：深度优先搜索顾名思义，就是以深度为优先进行搜索，也就是"一条路走到黑"

2.算法设计：递归思想，每一次操作都可以看做是同一种方法的不同状态，就可以设计成递归，对DFS而言，每次对一个节点的处理就是处理该节点的左侧节点和右侧节点（也就是把该节点看成新的根节点），因此，DFS可以用递归的方式实现，当然，也能使用非递归形式的栈

###### 二叉线索树

1.基本概述：

DFS导致的时间复杂度增高是并不想被看到的，因此我们采用空指针域存储某种顺序遍历(前中后)的各个节点之间的关系

2.具体方法(以中序遍历为例子)：

2.1 我们需要区分究竟是线索还是指向孩子的指针域，所以给每个指针域增加一个状态变量 l_tag 和  r_tag 用于区分，此时left指向前驱元素 right指向后继元素

2.2 只有空指针域才可作为线索域，需要记录的是元素(某种方式)遍历的前驱和后继关系

2.3 中序遍历先处理左子树，再处理自身，再处理右子树

2.4 对于自身节点的处理，由于我们需要的是前驱和后继的关系，于是我们引入两个指针，一个是prev，记录当前节点的（中序遍历的）上一个节点，如果上一个节点的右指针域为空，就可以修改右指针域(后继指针域)，指向当前节点node，如果当前节点的左指针域为空，就可以修改该节点的左指针域(前驱指针域)，按照这种方式，就可以实现线索化了

2.5线索遍历：见代码注释

3.代码实现：

```java
package BinaryTree;
/*
线索二叉树
 */
class ThreadedBTree
{
    TreeNode root;//根节点
    int count;
    //节点定义
    class TreeNode
    {
        char data;
        TreeNode left, right;
        int  l_tag;//左标识符,0表示指向子树,1表示指向前驱元素
        int  r_tag;//右标识符,0表示指向子树,1表示指向后继元素
        TreeNode()
        {
            this.l_tag = 0;
            this.r_tag = 0;
        }
        /* 插入逻辑 */
        public void insert(TreeNode left,TreeNode right, char data)
        {
            this.data = data;
            this.left = left;
            this.right = right;
            count++;
        }
    }
    /* 对节点的访问 */
    public void visitNode(TreeNode node)
    {
        System.out.print(node.data+" ");
    }
    /* 初始化一个测试用的树 */
    public void initTestTree()
    {
        TreeNode NA = new TreeNode();
        TreeNode NB = new TreeNode();
        TreeNode NC = new TreeNode();
        TreeNode ND = new TreeNode();
        TreeNode NE = new TreeNode();
        TreeNode NF = new TreeNode();
        TreeNode NG = new TreeNode();
        TreeNode NH = new TreeNode();
        TreeNode NK = new TreeNode();
        this.root = NA;
        NA.insert(NB,NE,'A');
        NB.insert(null,NC,'B');
        NC.insert(ND,null,'C');
        ND.insert(null,null,'D');
        NE.insert(null,NF,'E');
        NF.insert(NG,null,'F');
        NG.insert(NH,NK,'G');
        NH.insert(null,null,'H');
        NK.insert(null,null,'K');
    }
    /* 线索化 */
    TreeNode prev;//前一节点
    public void inOrdering(TreeNode node)
    {
            if(node == null)
                return;
            inOrdering(node.left);
            if(node.left==null)//有空位
            {
                node.left = prev;
                node.l_tag = 1;//变成线索
            }
            if(prev!=null&&prev.right==null)//有空位
            {
                prev.right = node;
                prev.r_tag=1;//变成线索
            }
            prev=node;
            inOrdering(node.right);
    }
    /* 中序线索化后继遍历 */
    public void inOrdered()
    {
        TreeNode node = root;
        //找出中序遍历的第一个元素,第一个线索化的节点就是第一个遍历的节点
        //因为如果线索化了,那么必然没有线索化之前的该节点的左孩子为空，那么必然就是第一个访问的节点
        while(node!=null&&node.l_tag!=1)
        {
            node=node.left;
        }
        //找后继
        while(node!=null)
        {
            visitNode(node);
            //如果右侧就是线索化
            if(node.r_tag==1)
            {
                node=node.right;
            }
            //如果右侧指向子树 找右侧子树中中序遍历的第一个节点,也就是左侧第一个线索化的节点,也就是当前节点访问后的下一个节点
            else
            {
                node=node.right;
                while(node!=null&&node.l_tag!=1)
                {
                    node=node.left;
                }
            }
        }
    }
    /* 中序线索化前驱遍历 */
    /* 论断：中序遍历的最后一个节点必然没有右孩子 */
    public void inOrdered2()
    {
        TreeNode node = root;
        //找出中序遍历的最后一个元素,一直往右找,直到找到没有右孩子为止
        //充分性：中序遍历总是优先访问左子树，然后访问当前节点，最后访问右子树。因此，要找到最后一个被访问的节点，需要找到一个节点，在访问它之后没有其他节点需要访问。如果一个节点有右孩子，那么访问该节点后还会访问其右子树，因此最后一个节点必须没有右孩子。
        //必要性：而现在要尽可能拖延访问,也就是说一直往右下走(越右越后),除非遇到第一个没有右孩子的节点,那么就必须要访问了(这是在左子树已经被访问的前提下)
        while(node!=null&&node.right!=null)
        {
            node=node.right;
        }
        if(node!=null)
            node.r_tag=1;
        while(node!=null)
        {
            visitNode(node);
            if (node.l_tag == 1)//左侧是线索
            {
                node = node.left;
            }
            //左 根 右
            else//左侧不是线索,找左侧子树的中序遍历的最后一个节点 因为左边遍历完成之后才到当前节点,也就是当前节点访问前的上一个节点
            {
                node = node.left;
                while (node != null && node.r_tag != 1)
                {
                    node = node.right;
                }
            }
        }
    }

    public static void main(String[] args)
    {
        ThreadedBTree tree = new ThreadedBTree();
        tree.initTestTree();
        tree.inOrdering(tree.root);
        tree.inOrdered2();
    }
}
```
